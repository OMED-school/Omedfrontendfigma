Getting Started

  1. Fork the repository
  2. Clone your fork: git clone https://github.com/YOUR_USERNAME/Omedfrontendfigma.git
  3. Create a branch: git checkout -b feature/your-feature-name
  4. Make your changes
  5. Test thoroughly
  6. Commit: git commit -m "Add feature description"
  7. Push: git push origin feature/your-feature-name
  8. Create a Pull Request

Development Setup

npm install npm run dev

Code Style

  • Use TypeScript for type safety
  • Follow existing code patterns
  • Write meaningful commit messages
  • Add comments for complex logic
  • Keep components focused and reusable

Pull Request Guidelines

  • Describe what your PR does
  • Reference related issues
  • Include screenshots for UI changes
  • Ensure all tests pass
  • Keep PRs focused on one feature

Reporting Issues

When reporting bugs, please include:

  • Clear description of the issue
  • Steps to reproduce
  • Expected vs actual behavior
  • Screenshots if applicable
  • Browser/device information

Community Guidelines

  • Be respectful and inclusive
  • Help others learn and grow
  • Provide constructive feedback
  • Follow the code of conduct

Questions?

Feel free to open a discussion on GitHub or contact the maintainers.
