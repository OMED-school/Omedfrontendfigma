Key Features

01

User Authentication

Secure registration, login, and profile management with role-based access control for students, teachers, and principals.

02

Idea Submission & Voting

Students can submit ideas with categories and descriptions. Community members can upvote or downvote to show support.

03

Nested Comments

Rich discussion threads with nested replies, allowing for detailed conversations about each idea.

04

Review Workflows

Multi-stage approval process: Student submission ‚Üí Teacher review ‚Üí Principal approval with budget and timeline.

05

Real-time Chat

Built-in messaging system for communication between users, with real-time message delivery.

06

Progressive Web App

Installable on any device, works offline, and can be packaged as native iOS and Android apps using Capacitor.

Getting Started

1

Clone the Repository

git clone https://github.com/OMED-school/Omedfrontendfigma.git cd Omedfrontendfigma
2

Install Dependencies

npm install
3

Set Up Supabase Backend

Follow the complete guide in SUPABASE_SETUP.md

  • Create a Supabase project
  • Run database schema SQL
  • Configure authentication
  • Set up Row Level Security
  • Update project credentials
4

Start Development Server

npm run dev

Open http://localhost:3000 in your browser

5

Build for Production

npm run build

Production files will be in the /build directory
